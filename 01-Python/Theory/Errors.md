# IndexError

An IndexError occurs when you try to access an index that does not exist in a string, list, or other sequence types.

&nbsp;

```py
text = "Hi"
print(text[5])  # ‚ùå IndexError
```

&nbsp;

### Output:

```md
IndexError: string index out of range
```
